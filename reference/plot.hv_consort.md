# Render a CONSORT flow diagram

Draws the grid-based diagram stored in an `hv_consort` object on a new
graphics page.

## Usage

``` r
# S3 method for class 'hv_consort'
plot(x, ...)
```

## Arguments

- x:

  An `hv_consort` from
  [`hv_consort()`](https://ehrlinger.github.io/hvtiPlotR/reference/hv_consort.md).

- ...:

  Ignored; present for S3 consistency.

## Value

`x`, invisibly.
