# Theme for generating poster figures through powerpoint

Theme for generating poster figures through powerpoint

## Usage

``` r
hvti_theme_poster(
  base_size = HVTI_THEME_POSTER_BASE_SIZE,
  base_family = "",
  header_family = NULL,
  base_line_size = base_size/22,
  base_rect_size = base_size/22,
  ink = "black",
  paper = "white",
  accent = "#3366FF"
)
```

## Arguments

- base_size:

  the base font size

- base_family:

  base font family

- header_family:

  = NULL,

- base_line_size:

  = base_size / 22,

- base_rect_size:

  = base_size / 22,

- ink:

  = "white",

- paper:

  = "transparent",

- accent:

  = "#3366FF"
